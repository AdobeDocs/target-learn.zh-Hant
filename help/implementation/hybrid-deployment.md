# Adobe Target混合部署

Adobe Target公司為個人化提供獨特的混合部署模型——混合用戶端和伺服器端建置。 此混合模型可讓非技術使用者使用WYSIWYG Visual Experience Composer來製作實驗或個人化活動，並讓伺服器端執行、傳送和轉譯體驗，以提供高效能的發佈。 

## 目標

在本視訊結束前，您將：

* 瞭解Adobe Target的混合部署帶來什麼
* 瞭解混合部署的優點
* 觀察混合部署可能減少的伺服器呼叫
