# Adobe Target混合部署

Adobe Target提供獨特的混合部署模型，整合使用者端和伺服器端實作，實現個人化體驗。 此混合模型可讓非技術使用者使用WYSIWYG Visual Experience Composer編寫實驗或個人化活動，並讓伺服器端執行、傳送及轉譯體驗，以進行高效能傳遞。

## 目標

在本影片結束時，您將：

* 瞭解Adobe Target混合部署的要求
* 瞭解混合部署的優點
* 觀察混合式部署可減少的伺服器呼叫
