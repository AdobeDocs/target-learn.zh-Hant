# Adobe Target 教學課程

[!DNL Adobe Target] 是 [!DNL Adobe Experience Cloud] 解決方案，提供一切所需工具，讓您量身訂造及個人化您的客戶體驗。 [!DNL Target] 協助您在網站和行動網站、應用程式、社交媒體和其他數位頻道上獲得最大收入。使用這些影片和教學課程來學習的許多元件 [!DNL Adobe Target].
