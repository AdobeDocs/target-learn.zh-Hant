# 在[!DNL Adobe Target]中設定屬性

## 說明

在本影片中，您將瞭解如何：

* 在[!DNL Adobe Target]中建立屬性

## 目標對象

* 管理員管理員

## 要求

使用者必須具備下列許可權才能建立屬性：

* 產品管理員（不要與產品設定檔管理員混淆）
* 系統管理員
