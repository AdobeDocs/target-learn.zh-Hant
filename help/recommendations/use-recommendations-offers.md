# 在A/B測試和體驗鎖定活動中使用Recommendations選件

Recommendations選件可搭配A/B測試和體驗鎖定活動使用。 此功能開啟了豐富的功能，讓您能夠：

* 在同一個活動內混合建議和非建議內容
* 輕鬆試驗建議在頁面上的放置方式，包括多個建議的順序
* 使用自動分配將流量推送到績效最好的建議體驗
* 使用自動鎖定目標，根據訪客的設定檔，以動態方式為訪客指派自訂建議體驗
