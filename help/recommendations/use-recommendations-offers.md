# 在A/B測試和體驗鎖定目標活動中使用Recommendations選件

Recommendations選件可與A/B測試和體驗鎖定目標活動搭配使用。 此功能會開啟豐富的功能，讓您：

* 在相同活動中混合建議和非建議內容
* 輕鬆試驗建議在頁面上的放置方式，包括多個建議的順序
* 使用自動分配將流量推送至績效最佳的建議體驗
* 使用自動鎖定目標，根據訪客的設定檔，以動態方式為訪客指派自訂建議體驗
