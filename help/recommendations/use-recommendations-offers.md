# 在A/B測試和體驗定位活動中使用Recommendations選件

Recommendations優惠可搭配A/B測試和體驗定位活動使用。 此功能可開啟豐富的功能，讓您：

* 在同一活動中混合建議和非建議內容
* 輕鬆嘗試在頁面上放置建議，包括多個建議的順序
* 使用自動分配功能，將流量自動推送至效能最佳的建議體驗
* 使用Auto-Target，根據訪客的個人檔案，動態指派訪客至量身打造的建議體驗
