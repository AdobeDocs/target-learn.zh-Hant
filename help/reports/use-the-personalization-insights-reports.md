# 使用個人化見解報表

「自動個人化」和「自動定位」活動使用進階機器學習，根據每位訪客的個人客戶個人檔案以及先前具有類似個人檔案的訪客的行為，為每位訪客提供最自訂的體驗。 「個人化洞察」報表提供這些模型如何做出決策的資訊。
