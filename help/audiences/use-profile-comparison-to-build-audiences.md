# 使用設定檔比較在Adobe Target中建立對象

Adobe Target中的[!UICONTROL 設定檔比較]功能可讓您在建立受眾時，彼此比較兩個數值設定檔值。 舉例來說，如果您將自訂建置的傾向分數傳入[!DNL Target]，並且想要根據傾向分數最高的產品來個人化內容，這個功能會非常實用。

## 說明

此影片會說明如何：

* 比較傳遞至Adobe Target的數值設定檔值（分數）
* 使用這些分數比較來定義對象
