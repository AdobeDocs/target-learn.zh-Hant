# 在Adobe Target中使用設定檔比較來建立對象

Adobe Target中的[!UICONTROL Profile Comparison]功能可讓您在建立受眾時比較兩個數值設定檔值。 如果您想要將自訂建立的傾向分數傳入[!DNL Target]，並想要根據傾向分數最高的產品來個人化內容，這會非常實用。

## 學習目標

在本影片中，您將瞭解如何：

* 比較傳遞至Adobe Target的數值設定檔值（分數）
* 使用這些分數比較來定義對象
